introdução ao Html

o html é uma linguagem de marcação para a internet.
É a linguagem "base" para a internet, necessaria para definir onde cada elemento (textos,imagens ou videos,etc) deve estar.


o navegador (chrome,fire fox, safari,etc) "lê" o código digitado e o "interpreta" exibindo assim na tela o site ja formatado e não códigos e TAGS.


OCSS

O CSS folha de setilo em cascatas para formatar as tags definidas no Html.

as formatações incluem cor, altura, largura,alinhamento, aparência de objetos em geral.

Principais TAGS

  • A tag <p> serve para definir parágrafo.
  • a tag <br>serve para quebrar a linha
  • a tag <h1>serve para definir o título e mostrar aos buscadores a importância do título.
  • a tag <ul>serve para fazer um lista desordenada.
  • a tag <li>serve para adicionar elementos na lista.
  • a tag font-sizeserve para redefinir o tamanho da fonte.
  • a tag font-familyserve para definir o estilo de fonte que será incorporado ao texto.
  • a tag background-colorserve para definir a cor de fundo do texto.
  • a tag colorserve para trocar a cor do texto
